# Simply Chocolate Website

Веб-сайт для компанії Simply Chocolate, створений на основі дизайну з Figma.

## Структура проекту

```
green-palet/
├── index.html          # Головна HTML сторінка
├── styles.css          # Стилі CSS
├── script.js           # JavaScript для інтерактивності
├── public/
│   └── images/         # Зображення з Figma
└── README.md           # Документація
```

## Особливості

- ✅ Повна адаптивність для різних розмірів екранів
- ✅ Плавна прокрутка по навігації
- ✅ Анімації при прокрутці сторінки
- ✅ Інтерактивні елементи (кнопки, картки продуктів)
- ✅ Сучасний дизайн відповідно до макету Figma

## Як запустити

1. Відкрийте файл `index.html` у браузері
2. Або використайте локальний сервер:

```bash
# З Python
python -m http.server 8000

# З Node.js (якщо встановлено http-server)
npx http-server

# Потім відкрийте http://localhost:8000
```

## Секції сайту

1. **Header** - Навігація та логотип
2. **Hero** - Головний банер з призивом до дії
3. **Benefits** - Переваги продукту
4. **Ingredients** - Інгредієнти шоколаду
5. **How It's Made** - Процес виготовлення
6. **Products** - Каталог продуктів
7. **Top Sellers** - Топ продажів
8. **Reviews** - Відгуки клієнтів
9. **Subscribe** - Підписка на новини
10. **Footer** - Футер з контактами

## Технології
